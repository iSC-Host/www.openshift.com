The power of OpenShift Online - customized for your workload

Dedicated Node Services FAQ

What is an OpenShift Online node?
A node is a virtual machine running on Amazon Web Services. A node contains one or more gears.

What is a gear?
A gear is a secure container for your code with allocated CPU, memory, disk, and network bandwidth.

What are the limits for custom gear sizes?
The 2vCPU node gear profiles can have up to 7.5GiB of memory and the 4vCPU node gear profiles can have up to 15GiB of memory.

What level of support do I receive when purchasing OpenShift Dedicated Node Service?
All OpenShift Dedicated Node Services offer Red Hat Professional Support.

If I purchase multiple nodes, can they be different sizes?
Yes. For example, you could purchase one 2vCPU node and one 4vCPU node.

Can I buy Dedicated Node Services with my credit card?
At this time Dedicated Node Services are only available through our regular sales channels. Please contact us and a sales person will help guide you through the process.

Are Dedicated Node Services available in the same countries as other OpenShift plans?
Dedicated Node Services are available for purchase worldwide.

How many users can a Dedicated Node Services plan support?
Each plan is assigned an administrator. This administrator can add any number of users using the Teams feature of OpenShift Online. Each plan supports up to 15 teams and 100 members per team.
